# SEO Content Recommendations

These suggestions are intentionally not implemented because Phase 2 restricts changes to technical on-page SEO and crawlability. They require manual content and UX review.

## Heading semantics

- Review the homepage H1 so it describes Vednovaa's placement readiness, AI/ML, and cybersecurity offering rather than functioning primarily as promotional card copy.
- Review the Services page H1 markup so the complete visible phrase is represented by one meaningful heading.
- Review the Case Studies page H1 so it clearly identifies the page as Vednovaa case studies or student outcomes.
- Review the Contact page H1 so it identifies Vednovaa partnership and training enquiries.

## Content and keyword coverage

- Consider adding concise, naturally written supporting copy around placement readiness programs, college partnerships, AI/ML training, cybersecurity simulations, and hackathons where those topics are already represented by the page.
- Consider adding visible introductory copy to the Case Studies page explaining the institutions, programs, and outcomes represented.
- Review case-study copy for consistent institution naming, program terminology, and measurable outcomes.

## Future content opportunities

- Evaluate dedicated landing pages only when Vednovaa has enough unique, useful information for each program and a clear user need. Do not create thin or duplicate pages solely for keywords.
- Consider a resource or blog strategy only if Vednovaa can maintain original, expert-led content for colleges, placement teams, and students.
- If a visible FAQ section is added in the future, use questions that reflect genuine institutional enquiries and add matching FAQ structured data only for the answers displayed on the page.

## Calls to action

- Review CTA wording in a future content phase for clearer intent by audience, such as colleges, placement officers, or students, while preserving the current conversion paths until approved.
